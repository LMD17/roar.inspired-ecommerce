# E-commerce Website

This is an E-commerce website project built using React for the frontend, Node.js for the backend, and a separate admin panel.

## File Structure

The project is structured as follows:

```
E-commerce
│
├── admin
│
├── backend
│
├── frontend
│
└── requirements.txt
```

## Getting Started

Follow the instructions below to set up and run the project.

### Prerequisites

- Node.js (version X.X.X or later)
- npm (version X.X.X or later)
- Python (for `requirements.txt`)

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/e-commerce.git
   cd e-commerce
   ```

2. Install the necessary libraries for the backend, frontend, and admin:

   - **Backend**:
     ```sh
     cd backend
     npm install
     ```

   - **Frontend**:
     ```sh
     cd frontend
     npm install
     ```

   - **Admin**:
     ```sh
     cd admin
     npm install
     ```

3. Install additional libraries listed in `requirements.txt`:
   ```sh
   pip install -r requirements.txt
   ```

### Running the Project

1. **Backend**:
   ```sh
   cd backend
   node .\index.js
   ```

2. **Frontend**:
   ```sh
   cd frontend
   npm start
   ```

3. **Admin**:
   ```sh
   cd admin
   npm run dev
   ```
